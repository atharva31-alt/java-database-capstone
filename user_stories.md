## Admin User Stories
* As an Admin, I want to log into the system securely so that I can access the management dashboard.
* As an Admin, I want to create and update doctor profiles so that the clinic roster remains accurate.
* As an Admin, I want to assign specific access roles to staff members so that data access permissions are controlled.
* As an Admin, I want to view comprehensive clinical reports so that I can track overall clinic performance.
* As an Admin, I want to deactivate user accounts for former staff so that the system remains secure.

## Patient User Stories
* As a Patient, I want to register an account with my personal details so that I can use the online portal.
* As a Patient, I want to view real-time doctor availability so that I can pick an optimal appointment slot.
* As a Patient, I want to book an appointment online so that I don't have to call the clinic directly.
* As a Patient, I want to view my upcoming consultation schedule so that I can prepare for my appointments.
* As a Patient, I want to access my digital prescription history so that I can buy my medications easily.

## Doctor User Stories
* As a Doctor, I want to securely log in to my dashboard so that I can see my daily schedule.
* As a Doctor, I want to view a patient's medical history so that I can provide an accurate consultation.
* As a Doctor, I want to update my weekly time availability slots so that patients don't misbook me.
* As a Doctor, I want to write and submit dynamic digital prescriptions so that patients receive their orders instantly.
* As a Doctor, I want to see the total count of patients I have treated this month so that I can monitor my workload.
